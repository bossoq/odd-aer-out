<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Node

  ```sh
  brew install node
  ```

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/bossoq/odd-aer-out.git
   ```

2. To install dependencies & start server run

   ```sh
   yarn install
   yarn run dev
   ```
